# 水丞  

中國篆義，唐代，約九世紀任羅，章75厘米  

唐代早期青花陶器的罕有例子。水丞施敛片透明釉，釉下以军昆的藍與藍色無紋装饰，人聯想到湖南長沙窯褐綠陶瓷自由奔放的設計風格。水丞的造型，與唐代文人常用的三足筆洗和水丞相似。  

一九七五年，當唐代青花陶瓷碎片在江蘇省揚州市的沿岸城市被發掘出來後，才得以確源唐代已燒造青花陶瓷的事實。大部分估計產自河南南的章縣（今為章集市）的早期青花陶器和一些小碟，都是在一九九八年做一履沉没於印尼努里洞島附近海域，属於九世紀阿拉伯帆船「黑石」號打捞出來。這艘於亞洲水域發現到最古老的阿拉伯商船，承載六萬件中國陶瓷，包括如此水丞的器物。該船徙揚州出發，以東南亞及中東地區為目的地。
