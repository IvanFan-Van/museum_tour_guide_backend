# 微型碗  

泰國宋加洛·十五世紀或十六世紀青釉炻器，高4.3厘米HKU.C.1964.0583  

塑像、小罐、杯和瓶等微型器物多用於泰國本地儀式，隅爾與小型可彌式祭壇一同使用。正如這兩件青釉炻器，微型器物一般施淺線檀色碎片紋釉。底部則不施釉以露出褐紅色的陶胎。微型器物是宋加洛常見的青瓷產品，均為本地與外銷之用。
